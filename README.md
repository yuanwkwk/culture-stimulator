## 介绍

**文化人生模拟器** - 一款基于不同文化背景的人生模拟游戏，让用户体验不同文化下的人生故事。

### 功能特点

#### 核心功能
- **文化选择**：选择不同的文化背景开始人生模拟
- **人物属性系统**：学识、技艺、财富、声望、健康等基础属性，以及文化特有属性
- **人生阶段系统**：童年期、青少年期、成年期、晚年期四个阶段
- **事件生成引擎**：每个阶段随机生成事件，提供多个选择分支
- **结局系统**：根据属性和选择生成多重结局，从传奇到困顿
- **内容库**：查看所有文化模板和游戏统计数据
- **国际化支持**：支持中英文切换，语言偏好自动保存

#### 已实现内容
- **唐代文人**文化模板（完整可玩）
  - 21个精心设计的人生事件（童年4、青少年6、成年7、晚年4）
  - 10种不同的人生结局（传奇2、卓越2、良好1、普通3、困顿2）
  - 特殊属性：诗才、仕途
  - 涵盖科举、诗歌、官场、人际关系等多个维度

- **19世纪法国画家**文化模板（完整可玩）
  - 18个精心设计的人生事件（童年3、青少年5、成年6、晚年4）
  - 8种不同的人生结局（传奇2、卓越2、良好1、普通2、困顿1）
  - 特殊属性：画技、艺术声望
  - 涵盖艺术流派、沙龙展览、印象派运动等历史事件

### 使用说明

1. **选择语言**：点击右上角语言切换按钮，选择中文或英文
2. **选择文化**：在首页选择一个文化背景
3. **开始人生**：从童年期开始，经历各种事件
4. **做出选择**：每个事件提供2-3个选择，影响属性和后续发展
5. **查看结局**：完成所有阶段后，根据属性获得相应结局
6. **再次体验**：可以重新开始，体验不同的人生轨迹

### 设计特色

- **温暖的历史色调**：使用米色、褐色、金色等体现文化底蕴
- **简约优雅的UI**：卡片式设计，清晰的信息层级
- **流畅的动画**：页面切换和元素出现都有精心设计的动画
- **响应式设计**：完美适配桌面和移动设备
- **国际化体验**：完整的中英文支持，语言偏好持久化保存

## 本地开发

### 如何在本地编辑代码？

您可以选择 [VSCode](https://code.visualstudio.com/Download) 或者您常用的任何 IDE 编辑器，唯一的要求是安装 Node.js 和 npm.

### 环境要求

```
# Node.js ≥ 20
# npm ≥ 10
例如：
# node -v   # v20.18.3
# npm -v    # 10.8.2
```

具体安装步骤如下：

### 在 Windows 上安装 Node.js

```
# Step 1: 访问Node.js官网：https://nodejs.org/，点击下载后，会根据你的系统自动选择合适的版本（32位或64位）。
# Step 2: 运行安装程序：下载完成后，双击运行安装程序。
# Step 3: 完成安装：按照安装向导完成安装过程。
# Step 4: 验证安装：在命令提示符（cmd）或IDE终端（terminal）中输入 node -v 和 npm -v 来检查 Node.js 和 npm 是否正确安装。
```

### 在 macOS 上安装 Node.js

```
# Step 1: 使用Homebrew安装（推荐方法）：打开终端。输入命令brew install node并回车。如果尚未安装Homebrew，需要先安装Homebrew，
可以通过在终端中运行如下命令来安装：
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
或者使用官网安装程序：访问Node.js官网。下载macOS的.pkg安装包。打开下载的.pkg文件，按照提示完成安装。
# Step 2: 验证安装：在命令提示符（cmd）或IDE终端（terminal）中输入 node -v 和 npm -v 来检查 Node.js 和 npm 是否正确安装。
```

### 安装完后按照如下步骤操作：

```
# Step 1: 下载代码包
# Step 2: 解压代码包
# Step 3: 用IDE打开代码包，进入代码目录
# Step 4: IDE终端输入命令行，安装依赖：npm i
# Step 5: IDE终端输入命令行，启动开发服务器：npm run dev -- --host 127.0.0.1
```
