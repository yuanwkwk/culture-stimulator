# 创作入口简化更新

## 更新内容

### 1. 首页创作卡片简化

**之前**：三个独立的创作入口按钮
- 创作文化模板
- 创作事件
- 创作结局

**现在**：单一的创作入口
- 只保留"创作文化模板"一个按钮
- 按钮更大更醒目
- 描述更清晰："一站式创作文化模板、人生事件和结局"

### 2. 创作中心页面重新设计

**之前**：三个并列的创作类型卡片

**现在**：
- 主创作卡片（创作文化模板）
  - 大卡片设计，突出主要功能
  - 显示三步创作流程
  - 大按钮"开始创作"
- 两个快捷链接卡片
  - 我的创作
  - 社区创作

## 设计理念

### 为什么简化？

1. **整合式创作流程**
   - 文化模板、事件和结局现在在一个页面中完成
   - 不需要分开的入口
   - 避免用户困惑

2. **更清晰的用户体验**
   - 单一入口，明确目标
   - 减少选择，降低认知负担
   - 流程更顺畅

3. **突出核心功能**
   - 创作文化模板是核心功能
   - 事件和结局是文化模板的一部分
   - 不应该分散展示

## 新的创作流程

### 用户视角

1. **发现创作功能**
   - 首页看到醒目的创作卡片
   - 或点击导航栏"创作"

2. **进入创作中心**
   - 看到主创作卡片
   - 了解三步创作流程
   - 点击"开始创作"

3. **完成整个创作**
   - 填写基本信息
   - 创作人生事件
   - 创作人生结局
   - 一次性提交

4. **查看创作**
   - 点击"我的创作"
   - 查看状态和统计

## 视觉改进

### 首页创作卡片
- **更大的按钮**：从小卡片变为全宽大按钮

- **主次分明**：主创作卡片占据主要位置
- **流程可视化**：显示三步创作流程
- **快捷访问**：底部两个快捷链接卡片
