# 用户创作功能快速开始

## 第一步：登录

1. 打开应用首页
2. 点击右上角"登录"按钮
3. 输入用户名（例如：张三）
4. 点击"登录"按钮

**提示**：如果想测试管理员功能，勾选"以管理员身份登录"

## 第二步：进入创作中心

1. 登录后，点击首页右上角"创作"按钮
2. 或者直接访问 `/create` 路径

## 第三步：创作文化模板

### 填写基本信息
1. 点击"创作文化模板"卡片
2. 填写以下信息：
   - **文化名称**：例如"宋代商人"
   - **文化描述**：详细描述这个文化背景
   - **时代**：例如"宋代（960-1279年）"
   - **地区**：例如"中国"

### 添加特殊属性
1. 默认有一个属性输入框
2. 填写：
   - **属性名称**：例如"商业头脑"
   - **属性描述**：例如"经商能力和商业眼光"
   - **初始值**：0-100之间的数字，例如40
3. 点击"添加属性"可以添加更多属性
4. 点击"移除属性"可以删除不需要的属性

### 提交创作
1. 检查所有信息是否填写完整
2. 点击"提交创作"按钮
3. 看到"创作提交成功，等待审核"提示
4. 自动跳转到"我的创作"页面

## 第四步：查看我的创作

### 访问方式
- 从创作中心点击"我的创作"按钮
- 或直接访问 `/my-creations` 路径

### 页面功能
1. **统计卡片**：显示总创作数、已通过、待审核、已拒绝的数量
2. **分类标签**：
   - 文化：查看所有文化模板创作
   - 事件：查看所有事件创作（待实现）
   - 结局：查看所有结局创作（待实现）
3. **创作列表**：
   - 显示标题、描述
   - 显示审核状态（待审核/已通过/已拒绝）
   - 可以删除自己的创作

### 审核状态说明
- 🟡 **待审核**：刚提交的内容，等待管理员审核
- 🟢 **已通过**：审核通过，其他用户可以看到和使用
- 🔴 **已拒绝**：审核未通过，只有你自己能看到

## 第五步：管理员审核（可选）

### 以管理员身份登录
1. 登出当前账号
2. 重新登录，勾选"以管理员身份登录"
3. 输入管理员用户名

### 审核内容（待实现）
- 查看所有待审核内容
- 批准或拒绝创作
- 查看创作详情

## 常见问题

### Q: 为什么我的创作状态一直是"待审核"？
A: 当前版本的审核功能还在开发中。你可以：
1. 以管理员身份登录
2. 使用数据库直接修改状态为"approved"
3. 等待后续版本实现完整的审核功能

### Q: 如何删除我的创作？
A: 在"我的创作"页面，每个创作右上角有删除按钮（垃圾桶图标），点击即可删除。

### Q: 创作的内容会立即在游戏中使用吗？
A: 不会。创作的内容需要：
1. 提交后状态为"待审核"
2. 管理员审核通过后状态变为"已通过"
3. 只有"已通过"的内容才会在游戏中显示

### Q: 可以修改已提交的创作吗？
A: 当前版本不支持修改。你可以：
1. 删除原创作
2. 重新创建一个新的

### Q: 忘记用户名怎么办？
A: 用户信息保存在浏览器的 localStorage 中。你可以：
1. 打开浏览器开发者工具（F12）
2. 进入 Application/Storage → Local Storage
3. 查找 `cultural_life_user` 键
4. 或者直接重新登录，创建新用户
