# 文化模板更新说明

## 更新概述

本次更新对现有的**唐代文人**模板进行了全面优化，并新增了**19世纪法国画家**文化模板。

## 一、唐代文人模板优化

### 1. 问题分析与改进

#### 原有问题：
- **事件数量不足**：部分阶段只有2个事件，选择性较少
- **事件分布不均**：童年期和晚年期事件偏少
- **缺少关键人生节点**：缺少婚姻、疾病、人际关系等重要事件
- **属性影响不够平衡**：某些选择的属性变化过于极端
- **结局类型单一**：缺少中间层次的结局

#### 改进措施：
- **增加事件数量**：
  - 童年期：2个 → 4个（新增"童年玩伴"、"初识诗词"）
  - 青少年期：3个 → 6个（新增"师友之谊"、"家道中落"、"婚姻大事"）
  - 成年期：3个 → 7个（新增"文坛论战"、"疾病缠身"、"边塞之行"、"权贵相邀"）
  - 晚年期：2个 → 4个（新增"培养后进"、"人生感悟"）
  - **总计：10个 → 21个事件**

- **丰富事件类型**：
  - 人际关系：童年玩伴、师友之谊、婚姻大事
  - 人生挫折：家道中落、疾病缠身、政治风波
  - 职业发展：边塞之行、权贵相邀、文坛论战
  - 人生总结：培养后进、著书立说、人生感悟

- **增加结局类型**：
  - 原有：7个结局
  - 新增：3个结局（边塞诗人、布衣诗人、仕途坎坷）
  - **总计：10个结局**

## 二、19世纪法国画家模板

### 1. 文化背景

**时代**：19世纪（1800-1900年）  
**地点**：法国巴黎  
**历史背景**：从古典主义到印象派，艺术革命风起云涌的伟大时代

### 2. 特殊属性

- **画技**（初始值30）：绘画技巧和艺术造诣
- **艺术声望**（初始值10）：在艺术界的名望

### 3. 事件设计（共18个）

#### 童年期（3个事件）
1. **艺术启蒙**
   - 初次接触绘画艺术
   - 选择：学画、自学、放弃
2. **家庭背景**
   - 家庭对艺术梦想的态度
   - 选择：坚持、妥协、放弃
3. **初次展览**
   - 第一次公开展示作品
   - 选择：精心准备、顺其自然、退缩

#### 青少年期（5个事件）
1. **美术学院**
   - 进入巴黎美术学院
   - 选择：刻苦学习、广交朋友、特立独行
2. **艺术流派**
   - 选择艺术方向
   - 选择：古典主义、浪漫主义、现实主义
3. **巴黎沙龙**
   - 首次参加沙龙展
   - 选择：保守作品、创新作品、放弃
4. **爱情与艺术**
   - 爱情对创作的影响
   - 选择：投入爱情、平衡、专注艺术
5. **经济困境**
   - 年轻画家的生存问题
   - 选择：接受委托、坚持创作、寻找赞助

#### 成年期（6个事件）
1. **印象派运动**
   - 艺术革命的选择
   - 选择：加入、中立、反对
2. **个人画展**
   - 举办个人展览
   - 选择：精心策划、广邀宾客、低调展出
3. **艺术批评**
   - 面对批评的态度
   - 选择：虚心接受、据理力争、一蹶不振
4. **海外机遇**
   - 国际化发展机会
   - 选择：前往、谨慎、婉拒
5. **艺术学院教职**
   - 教学与创作的平衡
   - 选择：接受、婉拒、兼职
6. **健康危机**
   - 健康与艺术的抉择
   - 选择：暂停、边养边创、不顾健康

#### 晚年期（4个事件）
1. **艺术遗产**
   - 处理艺术遗产
   - 选择：捐赠、出售、留给家人
2. **回顾展**
   - 卢浮宫回顾展
   - 选择：精心筹备、选择精华、低调处理
3. **艺术传承**
   - 指导年轻画家
   - 选择：倾囊相授、择优而教、保守秘诀
4. **人生落幕**
   - 人生最后时光
   - 选择：继续创作、整理作品、安享晚年

### 4. 结局设计（8个）

#### 传奇级（2个）
1. **印象派大师**
   - 条件：画技80+、艺术声望75+、技艺60+
   - 描述：改变艺术史的先驱

2. **学院派泰斗**
   - 条件：画技75+、艺术声望70+、学识65+、财富60+
   - 描述：古典主义的代表人物

#### 卓越级（2个）
3. **先锋艺术家**
   - 条件：画技75+、艺术声望60+、技艺70+
   - 描述：现代艺术的先驱

4. **成功的画家**
   - 条件：画技65+、艺术声望55+、财富50+
   - 描述：获得认可的艺术家

#### 良好级（1个）
5. **小有名气**
   - 条件：画技55+、艺术声望45+、财富40+
   - 描述：有一定知名度的画家

#### 普通级（2个）
6. **穷困的天才**
   - 条件：画技75+、艺术声望30-、财富20-、健康30-
   - 描述：生前未获认可的悲剧天才

7. **平凡画师**
   - 条件：画技45+、艺术声望30+、财富35+
   - 描述：普通的画师

#### 困顿级（1个）
8. **被遗忘的画家**
   - 条件：画技40+、艺术声望20+、财富25+、健康30+
   - 描述：才华被埋没

## 三、游戏平衡性优化

### 1. 属性影响调整
- 单次事件属性变化范围：-30 到 +30
- 避免过于极端的属性变化
- 增加更多需要权衡的选择

### 2. 事件触发条件
- 增加更多带requirements的事件
- 提高游戏策略性
- 让玩家的选择更有意义

### 3. 结局条件优化
- 增加中间层次的结局
- 让不同玩法都能获得合理的结局
- 避免结局条件过于严格或宽松

1. **更丰富的故事线**：每个阶段都有更多选择
2. **更真实的人生体验**：包含成功、失败、挫折、转折
3. **更高的可玩性**：不同选择导致完全不同的人生轨迹

5. **更好的平衡性**：各种玩法都有相应的结局
