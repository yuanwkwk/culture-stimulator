# 文化人生模拟器 - 使用指南

## 快速开始

### 1. 首页 - 选择文化背景
- 打开应用后，您会看到可用的文化模板
- 目前已实现：**唐代文人**
- 点击文化卡片开始游戏

### 2. 游戏页面 - 体验人生
- **左侧**：人物属性面板
  - 基础属性：学识、技艺、财富、声望、健康
  - 特殊属性：诗才、仕途（唐代文人专属）
- **右侧**：事件卡片
  - 阅读事件描述
  - 选择一个选项（每个选项会影响不同属性）
- **顶部**：阶段指示器
  - 显示当前处于哪个人生阶段

### 3. 人生阶段
游戏分为四个阶段，每个阶段3个事件：
1. **童年期（5-12岁）**：文化启蒙，家庭背景
2. **青少年期（13-20岁）**：学习成长，初试身手
3. **成年期（21-50岁）**：主要人生，事业发展
4. **晚年期（51岁+）**：人生总结，回顾往事

### 4. 结局页面 - 人生落幕
完成所有阶段后，系统会根据您的属性和选择生成结局：
- **传奇**：名垂青史（如：诗仙传世、一代名臣）
- **卓越**：功成名就（如：文坛宗师、隐逸高士）
- **良好**：小有成就（如：平凡一生）
- **普通**：平凡度日（如：怀才不遇）
- **困顿**：潦倒终生

### 5. 内容库
- 查看所有可用的文化模板
- 查看游戏统计数据
- 了解每个文化的游玩次数

## 游戏技巧

### 如何获得好结局？
1. **专注发展**：选择一个方向（如诗才或仕途）持续发展
2. **平衡属性**：不要忽视健康和财富
3. **把握机遇**：某些事件需要特定属性才能触发
4. **多次尝试**：不同的选择会带来完全不同的人生

### 唐代文人攻略
- **诗仙路线**：专注提升诗才和学识，选择文学相关事件
- **名臣路线**：专注提升仕途和声望，选择官场相关事件
- **隐士路线**：提升诗才和健康，远离官场纷争

## 数据说明

### 属性范围
- 所有属性范围：0-100
- 初始值：学识10、技艺10、财富30、声望10、健康80
- 特殊属性初始值根据文化而定

### 事件触发
- 某些事件需要满足特定属性要求才能触发
- 例如："初试诗才"需要诗才≥30

### 结局判定
- 系统会计算您的属性与各个结局条件的匹配度
- 选择匹配度最高的结局展示
