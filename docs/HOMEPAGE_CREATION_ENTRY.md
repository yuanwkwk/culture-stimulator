# 首页创作入口说明

## 新增功能

在首页文化模板列表上方添加了**创作入口快捷卡片**，方便用户快速访问创作功能。

## 功能特点

### 1. 醒目的卡片设计
- 使用渐变背景（primary/secondary色调）
- 带有主色调边框，突出显示
- 响应式设计，适配桌面和移动端

### 2. 三个创作入口按钮
每个按钮都包含：
- **图标**：视觉识别
  - 📄 文化模板（FileText图标，蓝色）
  - 📅 事件（Calendar图标，绿色）
  - 🏆 结局（Trophy图标，黄色）
- **标题**：创作类型名称
- **描述**：简短说明

### 3. 智能状态显示
- **已登录用户**：
  - 显示"发挥创意，创作新的文化模板、事件和结局，与社区分享你的作品"
  - 右上角显示"我的创作"按钮
  - 点击创作按钮直接进入对应创作页面
  
- **未登录用户**：
  - 显示"登录后可以创作和评价内容"
  - 底部显示提示："💡 登录后可以创作和评价内容"
  - 点击创作按钮会跳转到首页（提示登录）

### 4. 快捷访问
- **创作文化模板**：跳转到 `/create/culture`
- **创作事件**：跳转到 `/create/event`（待实现）
- **创作结局**：跳转到 `/create/ending`（待实现）
- **我的创作**：跳转到 `/my-creations`

## 交互流程

### 已登录用户
1. 用户打开首页
2. 看到创作入口卡片
3. 点击"创作文化模板"按钮
4. 直接进入文化模板创作页面
5. 填写表单并提交

### 未登录用户
1. 用户打开首页
2. 看到创作入口卡片（显示登录提示）
3. 点击右上角"登录"按钮
4. 输入用户名并登录
5. 返回首页，现在可以点击创作按钮

## 样式特点

### 卡片样式
- 边框：2px 主色调边框（20%透明度）
- 背景：渐变背景（primary 5%透明度 → secondary 5%透明度）
- 圆角：默认圆角
- 阴影：无（使用边框突出）

### 按钮样式
- 类型：outline变体
- 高度：自适应（py-4）
- 布局：垂直排列（flex-col）
- 对齐：左对齐（items-start）
- 悬停效果：
  - 边框变为主色调
  - 背景变为主色调（5%透明度）

### 图标颜色
- 文化模板：chart-1（蓝色系）
- 事件：chart-2（绿色系）
- 结局：chart-3（黄色系）

## 响应式设计

### 桌面端（≥768px）
- 三个按钮横向排列（grid-cols-3）
- 显示"我的创作"按钮
- 较大的间距（gap-4）

### 移动端（<768px）
- 三个按钮纵向排列（grid-cols-1）
- 隐藏"我的创作"按钮（可通过创作中心访问）
- 较小的间距（gap-3）

## 动画效果

- 卡片入场动画：
  - 初始状态：透明度0，向下偏移20px
  - 最终状态：透明度1，原位置
  - 延迟：0.2秒（在标题动画之后）
  - 持续时间：0.5秒

- 文化卡片动画：
  - 延迟增加0.3秒（在创作卡片之后）
  - 每个卡片依次出现（0.1秒间隔）

## 优势

1. **可发现性强**：用户一进入首页就能看到创作入口
2. **操作便捷**：一键直达创作页面，减少点击次数
3. **视觉吸引**：渐变背景和彩色图标吸引用户注意
4. **状态感知**：根据登录状态显示不同内容
5. **引导明确**：未登录用户看到登录提示，引导注册
