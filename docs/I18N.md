# 国际化（i18n）功能说明

## 概述

文化人生模拟器现已支持完整的中英文切换功能，用户可以在界面上随时切换语言，语言偏好会自动保存到本地存储。

## 功能特性

### 1. 语言切换
- **位置**：所有页面右上角
- **图标**：🌐 Languages 图标
- **支持语言**：
  - 🇨🇳 中文（简体）
  - 🇺🇸 English（英语）

### 2. 自动保存
- 用户选择的语言会自动保存到浏览器的 localStorage
- 下次访问时自动应用上次选择的语言
- 默认语言：中文

### 3. 翻译覆盖范围

#### 界面文本
- 所有页面标题和副标题
- 导航菜单和按钮
- 表单标签和提示信息
- 错误消息和通知

#### 游戏内容
- 属性名称（学识、技艺、财富、声望、健康、诗才、仕途）
- 人生阶段名称（童年期、青少年期、成年期、晚年期）
- 成就等级（传奇、卓越、良好、普通、困顿）
- 统计数据标签

#### 页面覆盖
- ✅ 首页（HomePage）
- ✅ 游戏页面（GamePage）
- ✅ 结局页面（EndingPage）
- ✅ 内容库页面（LibraryPage）

## 技术实现

### 架构设计

```
src/
├── locales/           # 翻译文件
│   ├── zh.ts         # 中文翻译
│   └── en.ts         # 英文翻译
├── contexts/
│   └── I18nContext.tsx  # 国际化上下文
└── components/
    └── common/
        └── LanguageSwitcher.tsx  # 语言切换组件
```

### 使用方法

#### 1. 在组件中使用翻译

```tsx
import { useI18n } from '@/contexts/I18nContext';

function MyComponent() {
  const { t } = useI18n();
  
  return (
    <div>
      <h1>{t.home.title}</h1>
      <p>{t.home.description}</p>
    </div>
  );
}
```

#### 2. 带变量的翻译

```tsx
import { useI18n, interpolate } from '@/contexts/I18nContext';

function MyComponent() {
  const { t } = useI18n();
  const cultureName = '唐代文人';
  
  return (
    <p>{interpolate(t.ending.subtitle, { culture: cultureName })}</p>
  );
}
```

#### 3. 切换语言

```tsx
import { useI18n } from '@/contexts/I18nContext';

function MyComponent() {
  const { language, setLanguage } = useI18n();
  
  return (
    <button onClick={() => setLanguage('en')}>
      Switch to English
    </button>
  );
}
```

## 添加新语言

如需添加新语言（如日语、法语等），请按以下步骤操作：

### 1. 创建翻译文件

在 `src/locales/` 目录下创建新的翻译文件，例如 `ja.ts`：

```typescript
import type { TranslationKeys } from './zh';

export const ja: TranslationKeys = {
  common: {
    back: '戻る',
    home: 'ホーム',
    // ...
  },
  // ... 完整翻译
};
```

### 2. 更新 I18nContext

在 `src/contexts/I18nContext.tsx` 中：

```typescript
import { ja } from '@/locales/ja';

type Language = 'zh' | 'en' | 'ja';

const translations: Record<Language, Translations> = {
  zh,
  en,
  ja,
};
```

### 3. 更新语言切换器

在 `src/components/common/LanguageSwitcher.tsx` 中添加新选项：

```tsx
<DropdownMenuItem
  onClick={() => setLanguage('ja')}
  className={language === 'ja' ? 'bg-accent' : ''}
>
  <span className="mr-2">🇯🇵</span>
  日本語
</DropdownMenuItem>
```

## 注意事项

### 1. 类型安全
- 所有翻译文件都使用 TypeScript 类型检查
- `en.ts` 必须实现 `TranslationKeys` 接口，确保与 `zh.ts` 结构一致

### 2. 翻译质量
- 当前英文翻译为机器翻译，建议由母语者审核
- 游戏内容（事件、结局）暂未翻译，仍为中文

### 3. 性能优化
- 翻译文件在应用启动时一次性加载
- 语言切换无需重新加载页面
- 使用 React Context 避免 prop drilling
