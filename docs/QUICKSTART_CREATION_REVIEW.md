# 快速开始：创作和审核文化模板

## 用户创作流程

### 步骤1：登录
1. 打开应用首页
2. 点击右上角"登录"
3. 输入用户名（例如：张三）
4. 点击"登录"

### 步骤2：进入创作页面
**方式1：从首页创作卡片**
- 在首页找到"创作中心"卡片
- 点击"创作文化模板"

**方式2：从创作中心**
- 点击导航栏"创作"
- 点击"创作文化模板"卡片

### 步骤3：填写基本信息
1. 切换到"基本信息"标签页
2. 填写：
   ```
   文化名称：宋代商人
   文化描述：宋代商业繁荣，商人地位提升...
   时代：宋代（960-1279年）
   地区：中国
   ```
3. 添加特殊属性：
   - 点击"添加属性"
   - 填写属性名称、描述、初始值
   - 至少添加1个属性

### 步骤4：创作人生事件
1. 切换到"人生事件"标签页
2. 选择人生阶段（童年期）
3. 点击"为童年期添加事件"
4. 填写事件信息：
   ```
   事件标题：家庭背景
   事件描述：你出生在一个商人家庭...
   ```
5. 添加选择项（至少2个）：
   ```
   选择1：跟随父亲学习经商
   选择2：专心读书科举
   ```
6. 重复步骤2-5，为其他阶段添加事件
   - 青少年期：至少1个事件
   - 成年期：至少1个事件
   - 晚年期：至少1个事件

### 步骤5：创作人生结局
1. 切换到"人生结局"标签页
2. 点击"添加结局"
3. 填写结局信息：
   ```
   结局标题：富甲一方
   结局描述：你成为宋代最富有的商人之一...
   成就等级：传奇
   ```
4. 重复步骤2-3，至少添加3个结局

### 步骤6：预览和提交
1. 点击右上角"预览"按钮
2. 检查创作统计：
   - 基本信息是否完整
   - 事件数量是否足够（至少4个）
   - 结局数量是否足够（至少3个）
3. 点击"编辑"返回修改（如需要）
4. 点击底部"提交创作"按钮
5. 等待提示"创作提交成功，等待审核"
6. 自动跳转到"我的创作"页面

### 步骤7：查看创作状态
1. 在"我的创作"页面
2. 查看创作状态：
   - 🟡 待审核（pending）
   - 🟢 已通过（approved）
   - 🔴 已拒绝（rejected）

## 管理员审核流程

### 步骤1：以管理员身份登录
1. 点击右上角"登录"
2. 输入用户名（例如：admin）
3. **勾选"以管理员身份登录"**
4. 点击"登录"
5. 用户名旁边会显示盾牌图标 🛡️

### 步骤2：进入审核页面
1. 点击右上角用户菜单
2. 选择"内容审核"
3. 或直接访问 `/admin/review`

### 步骤3：查看待审核内容
1. 查看统计卡片：
   - 待审核文化模板数量
   - 待审核事件数量
   - 待审核结局数量
2. 切换标签页查看不同类型的内容

### 步骤4：审核文化模板
1. 在"文化"标签页
2. 查看文化模板信息：
   - 文化名称
   - 时代和地区
   - 描述
   - 创作者
3. 点击"查看详情"查看完整信息
4. 评估内容质量
5. 做出决定：
   - 点击"批准"→ 内容对所有用户可见
   - 点击"拒绝"→ 内容仅创作者可见

### 步骤5：审核事件和结局
1. 切换到"事件"或"结局"标签页
2. 重复步骤4的审核流程
3. 注意：
   - 事件和结局是关联到文化模板的
   - 建议先审核文化模板，再审核事件和结局

## 验证清单

### 创作前检查
- [ ] 已登录
- [ ] 了解要创作的文化背景
- [ ] 准备好事件和结局的创意

### 提交前检查
- [ ] 基本信息完整（名称、描述、时代、地区）
- [ ] 至少1个特殊属性
- [ ] 每个人生阶段至少1个事件（共4个）
- [ ] 每个事件至少2个选择项
- [ ] 至少3个结局
- [ ] 所有必填字段都已填写

### 审核前检查
- [ ] 以管理员身份登录
- [ ] 了解审核标准
- [ ] 有足够时间审核

### 审核标准
- [ ] 内容符合文化背景
- [ ] 描述详细，有吸引力
- [ ] 事件和结局多样化
- [ ] 无不当内容
- [ ] 无抄袭或重复

## 常见错误和解决方案

### 错误1：提交失败
**原因**：表单验证未通过
**解决**：
1. 检查所有必填字段
2. 确保至少1个特殊属性
3. 确保每个阶段至少1个事件
4. 确保至少3个结局

### 错误2：看不到创作入口
**原因**：未登录
**解决**：
1. 点击右上角"登录"
2. 输入用户名并登录

### 错误3：无法访问审核页面
**原因**：不是管理员
**解决**：
1. 登出当前账号
2. 重新登录时勾选"以管理员身份登录"

### 错误4：创作的内容看不到
**原因**：内容还在待审核状态
**解决**：
1. 进入"我的创作"页面查看状态
2. 等待管理员审核
3. 或以管理员身份登录自行审核

## 提示和技巧

### 创作技巧
1. **先规划后创作**：在纸上或文档中先规划好整个文化背景
2. **参考现有模板**：查看官方模板获取灵感
3. **保持一致性**：确保事件和结局与文化背景一致
4. **多样化**：提供多种不同的事件和结局选择
5. **使用预览**：提交前使用预览功能检查

### 审核技巧
1. **批量审核**：一次性审核多个内容
2. **先看详情**：不确定时先查看详情
3. **记录标准**：建立自己的审核标准清单
4. **定期审核**：每天固定时间审核
5. **反馈机制**：未来版本会支持审核备注
